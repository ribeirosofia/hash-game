<h1 align="center>Hash-game</h1>
<p>To run the project it's pretty simple. Just open the terminal and it will run it</p>
<p>You have to digit the movement aligned with the 9 numbers of the board and which number remains null.</p>
<p>Then, you'll wait until the computer do its movement</p>
